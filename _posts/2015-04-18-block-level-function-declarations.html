Block-level Functions

{% markdown %} https://github.com/JustinDrake/node-es6-examples
On a more esoteric note, `ES6` is fixing a long standing issue with block scope function definitions. In ES6 function declarations are block-scoped. In particular, ES6 supports `block-level functions`, so the example code should produce the error “doSomething is not defined”. {% endmarkdown %}
{% highlight javascript linenos %} { function doSomething(){ console.log("In doSomething"); } } doSomething(); //doSomething is not defined {% endhighlight %}
