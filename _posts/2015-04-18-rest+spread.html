Rest + Spread

Rest parameters allows you to bind trailing paramaters into an array. To specify parameters as rest parameters, prepend the argument with three dots, as in line 1. Rest parameters are not included in a function's `length` property, illustrated on line 6.

{% highlight javascript linenos %} function f (x, ...y) { return x * y.length; // y is an array } f(3, "hello", true) // => 6 f.length // => 1 {% endhighlight %}

The spread operator turns an array into consecutive arguments in a function call. Each element in the passed array becomes a separate argument to the function. Spread operators also work with strings, as in line 6, and other iterable objects.

{% highlight javascript linenos %} function f (x, y, z) { return x + y + z; } f(...[1,2,3]) // => 6 f(..."abc") // => "a" + "b" + "c" ==== "abc" f.length // => 3 {% endhighlight %}
